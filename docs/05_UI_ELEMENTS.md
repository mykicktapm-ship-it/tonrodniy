# UI Elements

## Lobby Row

- Click on row:
  - Action: viewLobbyDetails
- Button "Join":
  - Action: joinLobby
- Button "Watch":
  - Action: watchLobby

## Confirmation Modal

- Confirm:
  - Action: sendTransaction
- Cancel:
  - Action: abortJoin

## Status Indicators

- playersCurrent / playersTotal
- lobby status
- countdown to deadline
