# TONRODY — Overview

TONRODY — это децентрализованное приложение на TON, предоставляющее
игровые лобби между пользователями без участия платформы в распределении средств.

## Ключевые принципы

- Платформа не участвует в игре и не получает комиссий.
- Все средства перемещаются только между пользователями.
- Смарт-контракт является единственным арбитром.
- Администратор не может:
  - отменить раунд,
  - изменить результат,
  - вывести средства игроков.
- Все правила зафиксированы в коде контракта.

## Основная механика

- Пользователи создают и заполняют лобби.
- После заполнения лобби начинается раунд.
- Победитель определяется контрактом.
- Весь пул переводится победителю (за вычетом газа сети).

## Типы лобби

- Classic Lobby
- Donate Lobby
- Experimental Lobby (включая Coin Flip как частный случай)
