The Trustless Gaming Protocol

TONRODY is a P2P gaming system on TON where the platform never plays against users, never takes a commission from game pools, and cannot manipulate outcomes.

verified_user Zero Fees

100% of the game pool goes to the winner. Revenue comes only from optional donations & customization.

gavel No Admin Keys

Contracts are immutable. Admins cannot cancel rounds, pick winners, or drain pools.

local_fire_department P2P Only

Funds move strictly Player ↔ Player. The platform is just the venue.

lock Critical Invariants

  • check_circle Game Pool = Sum(Stakes). In a 5-player lobby with 10 TON stake, the winner MUST receive exactly 50 TON.
  • check_circle No Platform Wallet in Game Flow. GameCore never sends game pool funds to Treasury. Only explicit donationAmount goes to Treasury.
  • check_circle Fail-safe Refunds. If a lobby expires or fails to fill, funds must be algorithmically refundable.
